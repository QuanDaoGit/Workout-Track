IRONBIT — INDICATOR ICONS
=========================

Three character-bar status glyphs, built in the pixel-arcade style
(muted core + emissive rim + soft bloom).

CONTENTS
--------
Indicator Icons (interactive showcase).html
    Self-contained, works offline. Open in any browser to see every
    icon, fill state, and variant rendered live at hero + small sizes,
    plus in-context character-bar / chip examples.

icons/   — transparent PNGs, 384×384, ready to drop into assets.

    stats-build.png            Stats / Build — neon-green radar triangle
    lck-streak.png             LCK / Streak — star + one magic orbit
    lck-streak-none.png        LCK variant — star only (no streak)
    lck-streak-hot.png         LCK variant — star + double orbit (hot streak)

    vitality-heart-full.png    Vitality — 100%
    vitality-heart-80.png      Vitality — 80%
    vitality-heart-60.png      Vitality — 60%
    vitality-heart-40.png      Vitality — 40%
    vitality-heart-20.png      Vitality — 20%
    vitality-heart-empty.png   Vitality — empty

COLOR LANGUAGE
--------------
Stats / Build    neon green  #00FF9C  (core #0E5236)
Vitality heart   arcade red  #E1283C / rim #FF4D63, navy shell #1C1C2A
LCK / Streak     amber star  #FFD700  +  violet magic  #B14DFF / #C77DFF

The PNGs are pixel-art with crisp edges — scale by integer multiples
(×2, ×3) for best results, or re-export at any size from the source
file "Indicator Icons.html" in the project root.

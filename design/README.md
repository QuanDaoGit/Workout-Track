# design/

Visual identity, UX conventions, and design references for Ironbit's pixel-arcade look.
Implementation truth is [../lib/theme/tokens.dart](../lib/theme/tokens.dart) — this folder
documents and references it, never forks it.

## Structure
- `brand.md` — palette, fonts, aesthetic, class colors.
- `ux-guidelines.md` — component / icon / motion / interaction rules.
- `screenshots/` — UI screenshots and visual audits.

See [CLAUDE.md](CLAUDE.md) for the working brief.

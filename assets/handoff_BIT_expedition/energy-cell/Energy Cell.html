ENERGY CELL — BIT'S EXPEDITION CHARGE

The fuel unit that gates SEND BIT. Each scouting run spends one cell; the HUD shows how many charges BIT has banked. Drawn as a sealed power cell in the dock's own pad-metal frame with a cyan glass core and a captured lightning glyph — BIT owns the cyan, so his fuel reads as his energy, never confused with the magenta haul currency.

Every cell is code-painted — plotted pixel-by-pixel on an 18×26 native canvas and blitted at an integer scale with nearest-neighbour, exactly like the pad / beam / haul cache. No CSS shapes, no upscaled PNG. Metal· glass· bolt

CHARGE STATES

Same frame, same bolt — only the glass tint and core brightness carry the level, so the silhouette is constant. DEPLETED drops the cyan entirely (dead grey glass, ghost bolt, no bloom) — absence reads as "needs recharge," never as failure. RECHARGING sweeps a bright fill-front up the glass.

IN THE HUD — CHARGE METER

Cells are the charge unit. Show banked charges as a row of pips (filled = charged, dead-grey = spent) with a mono count, seated beside the pad's send control.

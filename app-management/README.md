# app-management/

The app's lifecycle outside the code: roadmap, releases, store presence, support.

## Structure
- `roadmap.md` — product-level priorities & backlog.
- `releases/` — changelog, one file per version (`vX.Y.Z.md`).
- `store-listing/` — Play Store metadata & screenshots manifest.
- `support/` — FAQ, canned responses, known-issues log.

Engineering/build mechanics live in [../ops/](../ops/); marketing copy in [../marketing/](../marketing/).
See [CLAUDE.md](CLAUDE.md) for the working brief.

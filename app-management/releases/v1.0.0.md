# Ironbit v1.0.0+1 — First beta (Android)

> First external build. Distributed via Firebase App Distribution (signed APK,
> `com.erik.ironbit`). Pre-launch; no store listing yet.

## What's in this build

Everything — this is the initial beta of the full MVP:

- **Workout logging** — muscle-group start flow, curated exercise picker (favorites, filters),
  live session timer, per-set weight/reps logging, rest timer, summary with XP + stat deltas.
- **RPG layer** — STR/AGI/END radar with ranks D–S, VIT recovery meter, LCK streak multiplier,
  XP/levels/titles, 3 classes (Assassin/Bruiser/Tank) with focus bonuses.
- **Quests & gems** — auto-evaluated daily/weekly/side quests; earned-gem economy; cosmetic shop
  (frames/themes) + deterministic milestone loot.
- **Programs** — Full Body 3×, Upper/Lower, PPL; sequence-based 7-day cycles that prefill
  Start Workout; program path HUD and completion reveal.
- **Guild** — local single-player guild simulation, weekly cadence, Forge Nods.
- **Recovery** — planned rest days, shield charges, recovery XP, stat decay protection.
- **Body metrics (opt-in)** — body-neutral weight tracking, XP-boost potions.
- **Onboarding** — cinematic first-run: calibration quiz → name → class reveal → program select.
- **Exercise demos** — looping video demos for the FULL BODY A lifts with hide toggle.
- **Procedural avatars** — 20×20 pixel face customizer, zero image assets.

## Known issues (tracked in release-audit-2026-06-11.md)

- A process kill in the sub-second window during workout save can spend a potion charge without
  recording the session (S1).
- Streak/LCK and program day can misbehave if the device clock is set backwards (S4).

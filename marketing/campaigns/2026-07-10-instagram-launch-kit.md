## 2-week posting calendar

Best windows for a US/EU-leaning global audience: **Tue–Thu, 11:00–13:00 & 18:00–20:00** local-target.
Post reels in the evening window (higher completion), carousels midday.

| Day | Slot | Post | Notes |
|-----|------|------|-------|
| Wk1 Mon | eve | **P1 pinned intro** | Pin immediately after posting. |
| Wk1 Wed | eve | **P2 hero reel** | Boost with a trending-audio swap if reach is flat. |
| Wk1 Fri | midday | **P3 classes carousel** | Reply to every "my class" comment. |
| Wk1 Sun | eve | **P4 meme** | Lowest-effort, highest-share; good weekend filler. |
| Wk2 Tue | eve | **P5 feature reel** | |
| Wk2 Thu | midday | Devlog post/story | Erik first-person: a build-in-public update or beta metric. |
| Wk2 Sat | eve | Engagement filler | Poll story ("which class?") or a second meme. |

Daily (optional, low-lift): 1 story — a screenshot, a BIT reaction, or a "beta build #N shipped" note.
Stories keep the account warm between feed posts and drive the link sticker to the beta.

## Posting workflow (no API — manual / Business Suite)

1. **Account setup (once):** create @ironbit.app, set pfp + bio (from `marketing_memory.md`), add the
   bio link `https://ironbitbeta.netlify.app`, upload highlight covers. Switch to a **Creator account**
   (free) to unlock scheduling + insights.
2. **Schedule the batch:** open **business.facebook.com** (Meta Business Suite, free) → link the IG →
   *Create post* / *Create reel* → upload the finished asset from `posts/` → paste the caption →
   set the date/time from the calendar → Schedule. Repeat for all 5. One sitting.
3. **Reels spec:** 1080×1920, 9:16, ≤90s (ours are ~16s), cover frame = the RANK UP or wordmark frame.
4. **Carousel spec:** 1080×1350 (4:5, more feed real estate) — all slides same dimensions.
5. **After posting:** log each in the `marketing_memory.md` posting-log table; reply to comments in the
   first 60 min (Meta rewards early engagement).

# marketing/

Go-to-market material for Ironbit. Nothing here ships inside the app.

## Structure
- `positioning.md` — value prop, audience, key messages (start here).
- `copy/` — taglines, store/ASO copy, ad + social copy.
- `campaigns/` — dated campaign briefs.
- `assets/` — exported marketing images/banners.

Pre-launch status: building the launch story. See [CLAUDE.md](CLAUDE.md) for the working brief,
and [../docs/PRD.md](../docs/PRD.md) for what the product actually does.
